React-JS

Cuento con capacidad de crear, migrar, diseñar y maketar en react cualquier proyecto de la web, utilizando buenas prácticas, con conocimientos que van desde el uso de componentes de clase hasta el uso de componentes funcionales con hooks.

Redux

Integración de React con el store de redux, soy completamente capaz de utilizar estados locales de componentes o el store de redux si el proyecto así lo requiere, intento implementarlo en mis proyectos cada vez más, según los estándares y documentación.

React-Router

Ya que me enfoque en React intente he intento implementar poco a poco el stack completo, manejar rutas en SPAs o PWAs con react router, conociendo las ventajas y desventajas que se te tiene frente a otras opciones como nextjs y demás, me enfoco más en las rutas de react, para ser más a fin a la libreria.

Node-JS

Cuento con conocimientos en node y su administrador de paquetes, para crear configuraciones desde 0 a los proyectos de react, para utilizar código de JavaScript reciente, configuración de preprocesadores de estilos como SASS, crear REST APIs con JavaScript y más.

Java

Java como lenguaje por exelencia para el desarrollo de servidores, cuento con el conocimiento suficiente para desarrollar en este lenguaje usando buenas practicas de programación, implementación de pruebas unitarias en el código realizado y aplicando patrones de diseño y diferentes arquitecturas.

Sobre Mi

Desarrollador web

Mi nombre completo es Luis Eduardo Gamez Maldonado, nací el 12 de mayo del 2001 en Fundación- Magdalena, soy apasionado por la tecnología y el desarrollo de software, soy estudiante de ingeniería de sistemas en la universidad del magdalena, ubicada en Santa Marta Colombia, curso 6to semestre. Desde enero de 2019 incursioné en la web, aunque en la universidad aprendí programación estructurada en c++, estructuras de datos y demás, luego aprendí POO en java.

En ese tiempo un monitor me contacto y me hizo seguimiento porque “le gustaba como trabajaba el código y la idea que tenía para programar”... Digamos que fue quien me presento de lleno ante el mundo del desarrollo web. De ahí en adelante mis ganas de aprender y con mi capacidad autodidacta, ley, practiqué, seguí guías... Etc, todo lo necesario para entender cómo funcionaba la web, comencé a utilizar javascript como lenguaje principal. Una vez aprendi lo basico de la web, comenzaron a introducir conceptos como frameworks, librerias, entornos etc. Consulté a el monitor y me recomendó reactjs, entonces mi siguiente paso fue empezar en react, aprendí en un nivel intermedio a manejar componentes de clases, componentes funcionales, estados, propiedades, ciclos de vida, componentes presentacionales e inteligentes y lo que más pude, luego poco a poco fui integrando react con redux, implementando rutas y demas. Luego comencé a ver base de datos en la universidad y me pareció curioso como consultar esos datos desde un cliente de react, entonces comencé a aprender el funcionamiento de APIs, principalmente con el mismo lenguaje javascript en node, consumiendo de la base de datos que creaba en mssql-server en el curso de base de datos de la universidad.

Así a grandes rasgos aprendí el stack listado anteriormente en servicios, no estoy cerrado a un framework, ni a un tipo base de datos en especial, así como la tecnología avanza quienes la utilizamos debemos adaptarnos y no dejar de aprender, es así entonces que estoy abierto a otras opciones que me nutran como desarrollador web para escalar en mi experiencia.
