Servicios

Los servicios que puedo ofrecer son más que todo enfocados al front-end, pero también con conocimiento en creación de APIs y conexion con las mismas, me considero de nivel Junior hasta la fecha no cuento experiencia alguna, pero con toda la motivación y el empeño de aprender en el menor tiempo lo que me soliciten, a continuación listare los servicios o conocimientos que tengo acerca de la Web (El stack con el que tengo afinidad y complementos):

React-JS

Cuento con capacidad de crear, migrar, diseñar y maketar en react cualquier proyecto de la web, utilizando buenas practicas, con conocimientos que van desde el uso de componentes de clase hasta el uso de componentes funcionales con hooks.

Redux

Integración de React con el store de redux, soy completamente capaz de utilizar estados locales de componentes o el store de redux si el proyecto asi lo requiere, intento implementarlo en mis proyectos cada vez mas, segun los estandares y documentación.

React-Router

Ya que me enfoque en React intente he intento implementar poco a poco el stack completo, manejar rutas en SPAs o PWAs con react router, conociendo las ventajas y desventajas que se te tiene frente a otras opciones como nextjs y demas, me enfoco mas en las rutas de react, para ser mas a fin a la libreria.

SASS

En entornos mas profesionales seguro se implementara un preprocesador de css en vez del css puro, opte por aprender sass porque me parecio el mas completo, a futuro lo implementare con style-components, pero a mediano plazo puedo utilizar sass como preprocesaro de estilos.

Node-JS

Cuento con conocimientos en node y su administrador de paquetes, para crear configuraciones desde 0 a los proyectos de react, para utilizar codigo de JavaScript reciente, configuración de prepocesadores de estilos como SASS, crear REST APIs con JavaScript y mas.

Express

Al enfocarme en la Web tube que aprender a consumir datos de diferentes APIs, asi que pasado un tiempo entendi el como estaban construidas y que tecnologias se usaban, aprendi a manejar express, aunque no es fuerte intento no dejarlo de lado, conozco authenticación JWT, encriptacion de contraseñas con bcript etc.

Mssql-server

Tengo conocimiento intermedio en administración de bases de datos relacionales,aprendi en mssql-server pero puedo adaptarme a otras, concoimientos en diseño y optimización de DBS, procesos, vistas, procedimientos almacenados, etc. Cabe resaltar que tengo bases para funcionar en otros entornos como DBA.

Type-Script

Aunque no es comun que se implemente typescript en servicores, me parece una forma optima de sostener el codigo a travez del tiempo a parte de las caracteristicas que le da a javascript, dado que vengo de lenguajes como c++ y java que son de tipado fuerte y demas caracteristicas que lo hacen amigable a mi parecer.
